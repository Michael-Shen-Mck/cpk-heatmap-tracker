# 零基础云端网页部署步骤

目标：客户不安装 Python、不下载软件，只需要打开一个网址即可使用。

## 一、你需要准备

- 一个 GitHub 账号
- 一个 Streamlit Community Cloud 账号：https://share.streamlit.io
- 当前项目文件夹

## 二、上传到 GitHub

1. 在 GitHub 新建一个仓库，例如 `cpk-heatmap-tracker`。
2. 把本项目里的文件上传到这个仓库。
3. 不需要上传 `dist`、`__pycache__`、`.venv` 这些文件夹。

## 三、部署到 Streamlit Cloud

1. 打开 https://share.streamlit.io
2. 点击 `New app`。
3. Repository 选择刚上传的 GitHub 仓库。
4. Branch 选择 `main`。
5. Main file path 填：

```text
app.py
```

6. 点击 `Deploy`。
7. 等待部署完成后，会得到一个网址，例如：

```text
https://cpk-heatmap-tracker.streamlit.app
```

把这个网址发给客户即可。

## 四、客户怎么用

客户只需要：

1. 打开你发的网页链接。
2. 等待页面加载。
3. 直接录入、导入 Excel、查看热力图。

不需要安装 Python，不需要安装任何软件。

## 五、重要提醒

当前版本为了简单，未设置密码，知道网址的人都能打开。

当前版本使用 SQLite 文件数据库，适合演示、试点和轻量使用。云端免费平台可能在重启、休眠或重新部署后影响文件数据持久化。正式长期使用前，建议升级到云数据库，或定期在“数据明细与导出”中导出 CSV 备份。

## 六、演示数据清空

客户可以先用演示数据学习操作。正式开始录入前，进入“数据明细与导出”，打开“数据管理：删除演示数据或清空数据”：

- 点击“删除演示数据”：只删除 `DEMO-` 开头的演示批次，保留规格配置。
- 点击“清空全部捆重数据”：删除所有批次和捆重明细，保留规格配置。
