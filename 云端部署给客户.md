# 云端部署给客户使用

如果希望客户“点开一个网址就能直接使用”，推荐用 Streamlit Community Cloud。

## 操作步骤

1. 把本项目上传到 GitHub 仓库。
2. 打开 https://share.streamlit.io
3. 选择该 GitHub 仓库。
4. Main file path 填：

```text
app.py
```

5. 点击 Deploy。
6. 部署完成后，把生成的网址发给客户。

当前版本不设置密码，客户打开网址即可直接使用。

## 重要提醒

当前简单版使用 SQLite 文件数据库，适合演示和小范围试用。云端免费平台可能在重启后影响本地文件持久化，因此正式长期使用前建议升级为云数据库，或定期导出 CSV 备份。
