# CPK Heatmap 平台客户使用说明

## 一、怎么打开

1. 解压收到的压缩包。
2. 打开解压后的文件夹。
3. 双击 `启动平台.bat`。
4. 第一次启动会自动安装依赖，可能需要几分钟。
5. 浏览器打开后即可使用平台。

如果浏览器没有自动打开，请手动打开：

```text
http://localhost:8501
```

## 二、如果提示没有 Python

请先安装 Python：

1. 打开 https://www.python.org/downloads/
2. 下载 Windows 版本 Python。
3. 安装时勾选 `Add python.exe to PATH`。
4. 安装完成后重新双击 `启动平台.bat`。

## 三、日常使用流程

1. 在“规格配置”里确认规格、目标捆重、允许负差。
2. 在“数据录入”里选择规格、生产日期、轧线、班组和批号。
3. 可以手工粘贴捆重，也可以下载 Excel 模板后批量导入。
4. 在“过程能力热力图”里选择月份、轧线、班组查看 Cpl 热力图。
5. 在“数据明细与导出”里查看明细、导出 CSV 或删除错误批次。

## 四、Excel 导入说明

- 一行代表一捆重量。
- 同一批号可以填写多行，导入后会自动合并为一个批次。
- 轧线只能填写：`三轧`、`四轧`。
- 班组只能填写：`甲班`、`乙班`。
- 规格名称要和平台“规格配置”里的名称一致。

## 五、注意事项

- 关闭启动窗口后，平台会停止运行。
- 数据保存在本文件夹的 `data/cpk_tracker.db` 中，请不要删除 `data` 文件夹。
- 建议定期在“数据明细与导出”里导出 CSV 备份。
- 当前版本不需要密码，打开网页即可使用。
